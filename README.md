# How To Install
```
$ apt install git

$ apt install nodejs

$ git clone https://github.com/ALX-04/Konversi-Bilangan-CLI

$ npm install inquirer --save

$ npm install delay --save

$ npm install chalk --save

$ cd Konversi-Bilangan-CLI

$ node konversi.js
```
